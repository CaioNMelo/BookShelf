# BookShelf

BookShelf e um app Flutter estatico criado para a faculdade.

Ele representa a tela principal de um agregador de noticias, com:

- Tema claro com cor principal roxa
- Campo de busca
- Alternancia entre Brasil e Internacional
- Categorias com radio buttons
- Cards de noticias falsos para visualizar o layout

Ainda nao ha integracao com API. A tela foi criada apenas para demonstrar a interface.
